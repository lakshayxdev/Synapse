# 🧠 Synapse - AI Powered Notes Application

An AI-powered full-stack notes application built with the MERN stack that helps users create, organize, and summarize notes effortlessly. Synapse combines secure authentication, intelligent note management, and AI-generated summaries into a modern, responsive interface.

---

## ✨ Features

### 🔐 Authentication
- User Signup & Login
- JWT Authentication
- Protected Routes
- Secure Password Hashing

### 📝 Notes Management
- Create Notes
- Edit Notes
- Delete Notes
- View Full Notes
- Search Notes
- Responsive Notes Grid

### 📌 Organization
- Pin / Unpin Notes
- Save / Unsave Notes
- Archive / Unarchive Notes
- Filter Notes by Category

### 🤖 AI Integration
- Generate AI Summaries
- Loading Animation During Summary Generation
- Clean Bullet Point Summaries

### 🎨 User Experience
- Modern Dark UI
- Beautiful Landing Page
- Animated Components
- Toast Notifications
- Delete Confirmation Modal
- Empty State Screens
- Skeleton Loading
- Responsive Design
